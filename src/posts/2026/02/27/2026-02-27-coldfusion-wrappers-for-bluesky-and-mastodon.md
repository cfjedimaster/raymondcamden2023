It's been a hot minute since I opened a repo for ColdFusion code, but I thought I'd package up my previous wrapper for Mastodon support, port over my BoxLang Bluesky module, and properly release the code on GitHub for folks who want it: <https://github.com/cfjedimaster/coldfusion-social-wrappers>

Right now I'm using one repo for both Bluesky and Mastodon. Usually I'd separate them, but with the "support" being a grand total of one file each, I figured no one would mind getting the "extra" code if they only care about one. I'll also note the code is very much focused on *posting*, not reading data, but part of the reason I wanted to package this up was for people to run with it and add additional support. Fire off those PRs folks!
